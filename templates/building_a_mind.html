Building A Mind User Interface

Computer Vision Research and Product Design

The Building A Mind lab runs cutting-edge research on the nature vs nurture debate using computer vision, and it is opening up its research by making the project open-source. The software involved is complex, however, and limited documentation is available. To ensure the research is accessible to as many people as possible, I joined a team of other researchers to create a beginner-friendly interface and educational resource for the models.

Project Summary

Project At A Glance

  • Academic project
  • May 2024 - August 2024
  • Collaborators: Dr. Justin Wood, Dr. Samantha Wood, Lalit Pandey

Roles & Responsibilities

  • Computer Vision Research: Auditing the machine learning pipeline and planning the implementation of deep reinforcement learning.
  • Documentation: Discovering pain points in the installation process and updating documentation.
  • UX Design: Creating a web interface for documentation, research papers, and educational resources.

The Interface

The web interface provides an accessible way for users to explore the Building A Mind research and documentation. It features a clean, modern design that makes complex computer vision concepts approachable for beginners.

Project Overview

This project involved creating a comprehensive web interface that serves as both an educational resource and a gateway to the Building A Mind research. The interface includes documentation, research papers, and interactive elements to help users understand the computer vision models.
