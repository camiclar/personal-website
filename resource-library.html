Career Resource Library

Data Migration and Front-End Design

IU's office of Career Exploration & Student Employment hosts an online library of nearly 400 career resources and is regularly updated to add new resources and remove old ones. All of these resources were stored in a JSON file, however, making them difficult to edit, especially for those without technical backgrounds. This prompted a data migration project moving these resources to Airtable and updating the client-facing UI, creating a streamlined experience for both staff and students.

Project Summary

Project At A Glance

  • Professional project
  • June 2020 - November 2021
  • Collaborators: Stacey Abshire, Dr. Clinton Roeth-McKay

Roles & Responsibilities

  • Data Migration: Coordinating the transfer of data from JSON to Airtable.
  • Data Analysis: Applying statistical techniques to discover findings in the feedback.
  • Synthesizing Results: Reporting major insights from the data and suggesting next steps.

The Problem

The original resource library was difficult to maintain and update. All resources were stored in a JSON file, making it challenging for non-technical staff to add, edit, or remove resources. This created bottlenecks and limited the library's ability to stay current with new career resources.

Filtering Challenges

The original system had limited filtering capabilities, making it difficult for students to find relevant resources. The interface lacked intuitive ways to narrow down the nearly 400 resources by category, type, or other criteria.

The Solution

We migrated the data to Airtable, which provided a user-friendly interface for staff to manage resources, and redesigned the front-end to include improved filtering and search capabilities. The new system streamlined both the management and user experience.
